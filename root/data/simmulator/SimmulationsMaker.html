Wichtigstes zu JavaScript:

Variable Erstellen:

var variablenname = wert;
Beispiel: Erstellt Variable a und setzt auf 10.5:
var a = 10.5;

Variablen Inhalt verändern:

variablenname = neuerwert;
Beispiel: Setzt Variable a auf 11.5:
a = 11.5;

Rechnen:

Beispiel1: Addiert a und b und speichert den Inhalt in der neuen Variable c:
var c = a+b;
Beispiel2: Dividiert a und b und speichert den Inhalt in der neuen Variable c:
var c = a-b;
Beispiel3: Multipliziert a und b und speichert den Inhalt in der neuen Variable c:
var c = a*b;
Beispiel4: Dividiert a und b und speichert den Inhalt in der neuen Variable c:
var c = a/b;
Klammern können auch gesetzt werden: var c = (a+b)/c
Potenzieren: var potenz = Math.pow(basis,exponent);
Quadratwurzel: var wurzel = Math.sqrt(x);
Logarithmus (Basis e): var c = Math.log(potenz);
Betrag: var c = Math.abs(x);
Vorzeichen (wenn positiv 1, wenn negativ -1, wenn 0 dann 0: var c = Math.sign(x);
Winkelfunktionen (Bogenmaß): var sinus = Math.sin(x); var cosinus = Math.cos(x); var tangens = Math.cos(x);
Winkelfunktions Umkehrungen (Bogenmaß): var arcussinus = Math.asin(x); var arcuscosinus = Math.acos(x); var arcustangens = Math.acos(x);
Rest: var rest = zahl%teiler;

Konstaten:

Pi: var pi = Math.PI;
Eulersche Zahl: var e = Math.E;
Wurzel aus 2: var x = Math.SQRT2;
1/2 Wurzel aus 2: var x = Math.SQRT1_2;

Bedingungen:

Wenn a gleich b: a == b
Wenn a ungleich b: a != b
Wenn a größer b: a > b
Wenn a größergleich b: a >= b
Wenn a kleiner b: a < b
Wenn a kleinergleich b: a <= b
Wenn a gleich b und c gleich d: a == b && c == d
Wenn a gleich b oder c gleich d: a == b || c == d
Wenn a gleich b xoder c gleich d: a == b ^ c == d
Wenn Nicht (a gleich b): !(a == b)
